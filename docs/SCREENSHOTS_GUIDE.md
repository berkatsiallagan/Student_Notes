## üé® Screenshot Enhancement Tips

### Before Taking Screenshots

1. **Clean Device**
   - Clear all notifications
   - Set battery to 100% (or use battery icon remover)
   - Set time to 9:41 AM
   - Enable airplane mode (no signal bars)

2. **App Preparation**
   - Use good sample data
   - Ensure no lorem ipsum
   - Use realistic content
   - Check for typos

3. **Visual Polish**
   - Ensure good lighting (if physical device)
   - Clean screen (no fingerprints)
   - Proper orientation (portrait)

### After Taking Screenshots

1. **Crop & Resize**
   - Use exact dimensions required
   - Maintain aspect ratio
   - Remove status bar if needed

2. **Add Device Frame (Optional)**
   - Use tools like:
     - [Mockuphone](https://mockuphone.com/)
     - [Screely](https://www.screely.com/)
     - [Device Frames](https://deviceframes.com/)

3. **Add Captions (Optional)**
   - Use tools like:
     - Canva
     - Figma
     - Photoshop
   - Keep text readable
   - Use brand colors

4. **Optimize File Size**
   - Compress images
   - Use PNG for quality
   - Keep under 8 MB
