## üìö Tipografi Baru

### Font Family
**Inter** - Modern, clean, dan sangat readable

**Alasan Pemilihan:**
- Designed untuk UI/screen reading
- Excellent readability di semua ukuran
- Professional dan modern
- Open source dan gratis

### Typography Scale

#### Display (Headers Besar)
```
Display Large:  32px, Bold (700), -0.5 letter-spacing
Display Medium: 28px, Bold (700), -0.5 letter-spacing
Display Small:  24px, SemiBold (600), -0.25 letter-spacing
```

**Penggunaan:** Splash screen, welcome screens

#### Headline (Section Headers)
```
Headline Large:  22px, SemiBold (600), -0.25 letter-spacing
Headline Medium: 20px, SemiBold (600), 0 letter-spacing
Headline Small:  18px, SemiBold (600), 0 letter-spacing
```

**Penggunaan:** Page titles, section headers

#### Title (Card Titles)
```
Title Large:  18px, SemiBold (600), 0 letter-spacing
Title Medium: 16px, SemiBold (600), 0.1 letter-spacing
Title Small:  14px, SemiBold (600), 0.1 letter-spacing
```

**Penggunaan:** Card titles, list item titles

#### Body (Content Text)
```
Body Large:  16px, Regular (400), 0.15 letter-spacing, 1.6 line-height
Body Medium: 14px, Regular (400), 0.25 letter-spacing, 1.6 line-height
Body Small:  12px, Regular (400), 0.4 letter-spacing, 1.5 line-height
```

**Penggunaan:** Paragraphs, descriptions, content

#### Label (Buttons & Labels)
```
Label Large:  16px, SemiBold (600), 0.5 letter-spacing
Label Medium: 14px, SemiBold (600), 0.5 letter-spacing
Label Small:  12px, SemiBold (600), 0.5 letter-spacing
```

**Penggunaan:** Buttons, chips, badges, labels

### Typography Features
- **Letter Spacing:** Optimized untuk readability
- **Line Height:** 1.4-1.6 untuk comfortable reading
- **Font Weight:** Clear hierarchy (400, 600, 700)
- **Negative Tracking:** Untuk display sizes (tighter, modern look)

## üí° Design Rationale

### Why Green?
1. **Calming Effect:** Hijau memberikan efek tenang, cocok untuk belajar
2. **Natural:** Warna natural yang tidak melelahkan mata
3. **Fresh:** Memberikan kesan fresh dan energizing
4. **Professional:** Sage green adalah warna professional

### Why Inter Font?
1. **Readability:** Designed khusus untuk screen reading
2. **Modern:** Contemporary dan clean
3. **Versatile:** Bagus di semua ukuran
4. **Open Source:** Gratis dan legal

### Why Soft Colors?
1. **Eye Comfort:** Tidak harsh, nyaman untuk long sessions
2. **Professional:** Sophisticated, tidak childish
3. **Modern:** Muted colors adalah trend modern design
4. **Accessible:** Easier untuk color blind users
